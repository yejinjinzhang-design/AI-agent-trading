# CORAL Web Dashboard

Real-time monitoring dashboard for CORAL agent runs.

## Development

```bash
npm install
npm run dev      # Dev server with HMR
npm run build    # Production build (output → ../coral/web/static/)
npm run lint     # ESLint
```

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Server-Sent Events for live updates
